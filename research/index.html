Privacy-Preserving Vision-Based Detection of Pox Diseases Using Federated Learning
Published in Springer Nature · 2024
A federated learning framework for detecting pox-related skin diseases from visual data without centralising patient information. Each node trains locally — only weights are shared — preserving medical privacy while achieving competitive accuracy across heterogeneous data distributions.
